Documentation

The server-side component of this web application is being built from the ground up. The application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

Business Logic URL HTTP Method Request body data format Response body data format
Return a list of all movies to the user /movies GET None A JSON object holding data about all movies
Return data about a single movie by title to the user /movies/[Title] GET None A JSON object holding data about a single movie title. Example: 
{
  "Title": "Pulp Fiction",
  "Genre": {
    "Name": "Crime"
  },
  "Director": {
    "Name": "Quentin Tarantino"
  },
  "Year": "1994"
}
Return data about a genre by name/title /movies/genre/[Genre Name] GET None A JSON object holding data about a genre. Example: 
{
  "Name": "Drama",
  "Description": "Genre description goes here."
}
Return data about a director by name /movies/directors/[Director Name] GET None A JSON object holding data about a director. Example:
{
  "Name": "Quentin Tarantino",
  "Born": "March 27, 1963",
  "Bio": "Director bio goes here."
}
Allow new users to register /users POST A JSON object holding data about the user to be added. Example:
{
  "name": "Biff",
  "favoriteMovies": []
}
 A JSON object holding data about the user that was added, including an ID. Example:
{
  "name": "Biff",
  "favoriteMovies": [],
  "id": "1"
}
Allow users to update their user info (username) /users/[User ID] PUT A JSON object holding data about the user to be updated. Example:
{
  "name": "Biff McGee",
  "favoriteMovies": [],
  "id": "1"
}
 A JSON object holding the updated user data. Example:
{
  "name": "Biff McGee",
  "favoriteMovies": [],
  "id": "1"
}
Allow users to add a movie to their list of favorites /users/[User ID]/[Movie Title] POST None A text message indicating the movie was added to the user's favorites
Allow users to remove a movie from the list of favorites /users/[User ID]/[Movie Title] DELETE None A text message indicating the movie was removed from the user's favorites
Allow existing users to deregister /users/[User ID] DELETE None A text message indicating the user was deregistered
